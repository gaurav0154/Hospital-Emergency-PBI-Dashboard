🏥 Hospital Analytics Dashboard (Power BI)
📌 Project Overview

This project is an interactive Hospital Analytics Dashboard built using Power BI to analyze patient data, hospital operations, and performance metrics.

The dashboard provides deep insights into patient flow, wait times, department referrals, demographics, and operational efficiency, helping healthcare stakeholders make data-driven decisions.

🎯 Objectives
Analyze overall patient volume and trends
Monitor average wait time and service efficiency
Track patient admission vs non-admission rates
Understand department referral patterns
Study demographics (age, gender, race)
Evaluate hospital performance against targets
