# Hospital-Emergency-PBI-Dashboard
Power Bi Dashboard Hospital Emergency
🚑 Power BI Project – Hospital Emergency Room Dashboard
I worked on another Power BI project to strengthen my skills in data modeling, DAX calculations, and visualization techniques. In this dashboard, I applied different chart types and filters to analyze hospital emergency room data.
🔹 What I learned:
- How to use DAX for calculated measures and KPIs
- Importance of choosing the right chart for clear storytelling
- Translating raw data into meaningful insights for decision‑making
🔹 Why this project matters:
Hospital emergency rooms deal with critical situations where time, patient flow, and resource allocation are key. This dashboard highlights how data can improve operational efficiency, patient satisfaction, and healthcare outcomes.
